# 审核清单

## 内容审核

### 事实核查
- [ ] 数据来源是否可靠
- [ ] 引用是否准确
- [ ] 案例是否真实
- [ ] 时间信息是否正确

### 逻辑检查
- [ ] 论点是否有论据支撑
- [ ] 段落之间逻辑连贯
- [ ] 结论是否由前文推导
- [ ] 无自相矛盾之处

## 合规审核

### 敏感词检查
- [ ] 无政治敏感内容
- [ ] 无涉黄涉暴内容
- [ ] 无歧视性言论
- [ ] 无绝对化用语（最、第一、唯一等）

### 版权检查
- [ ] 图片有使用授权
- [ ] 引用已标注来源
- [ ] 无未授权转载内容

### 广告合规
- [ ] 广告内容真实
- [ ] 无虚假宣传
- [ ] 医疗/金融内容符合法规

## 平台适配检查

### 公众号
- [ ] 标题 ≤ 64 字
- [ ] 封面图比例正确
- [ ] 无诱导分享内容
- [ ] 排版整洁

### 小红书
- [ ] 标题 ≤ 20 字
- [ ] 正文 ≤ 1000 字
- [ ] 图片数量 3-9 张
- [ ] 话题标签 3-10 个
- [ ] 无外链/二维码

## 可读性检查

- [ ] 开头能吸引读者
- [ ] 段落长度适中
- [ ] 重点内容突出
- [ ] CTA 明确
- [ ] 结尾有力
